  1. Colors with MatLab like syntax

With matplotlib, we can define the colors of lines and other graphical elements in a number of ways. First of all, we can use the MATLAB-like syntax where 'b' means blue, 'g' means green, etc. The MATLAB API for selecting line styles are also supported: where, for example, 'b.-' means a blue line with dots:

In [4]:
# MATLAB style line color and style 

fig, ax = plt.subplots()
ax.plot(x, x**2, 'b.-') # 'b.-' -> Means Blue lines with Dots
ax.plot(x, x**3, 'g--') # 'g--' -> Means Blue with Dashed Line

plt.show()
  1. Colors with the color parameter

We can also define colors by their names or RGB hex codes and optionally provide an alpha value using the color and alpha keyword arguments. Alpha indicates opacity.

In [5]:
fig, ax = plt.subplots()

ax.plot(x, x+1, color="blue", alpha=0.5) # Half-Transparent
ax.plot(x, x+2, color="#8B008B")         # RGB hex code
ax.plot(x, x+3, color="#FF8C00")         # RGB hex code 

plt.plot()

  1. Line and marker styles

To change the line width, we can use the linewidth or lw keyword argument. The line style can be selected using the linestyle or ls keyword arguments:

In [6]:
fig, ax = plt.subplots(figsize=(12,6))

ax.plot(x, x+1, color="red", linewidth=0.25)
ax.plot(x, x+2, color="red", linewidth=0.50)
ax.plot(x, x+3, color="red", linewidth=1.00)
ax.plot(x, x+4, color="red", linewidth=2.00)

# Possible linestype: -, , -., :, steps
ax.plot(x, x+5, color="green", lw=3, linestyle='-')
ax.plot(x, x+6, color="green", lw=3, ls='-.')
ax.plot(x, x+7, color="green", lw=3, ls=':')

# Custom Dash
